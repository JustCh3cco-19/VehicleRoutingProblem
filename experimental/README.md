# Experimental material

This directory groups material that is useful for research and profiling, but
is not part of the production solver implementation.

- `exp_dist_calc/`: CPU/GPU distance-calculation microbenchmarks.
- `experiments/`: historical OpenMP/MPI implementation experiments.

Commands in the experiment scripts are intended to be run from the repository
root, so paths such as `src/`, `include/`, `instances/` and `experimental/`
resolve consistently.
